# Snip Project

This repository contains Jupyter notebooks for data science experiments.

## 📂 Contents
- **covid 19 trends.ipynb** – Analyzes COVID-19 case trends using public datasets and visualizations.
- **spam model.ipynb** – Builds and evaluates a machine learning model to classify spam messages.

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Jupyter Notebook

## 🚀 Getting Started
To run the notebooks locally:
1. Clone the repository:
   git clone https://github.com/Tommy278710/snip.git
2. Navigate into the folder:
   cd snip
3. Launch Jupyter:
   jupyter notebook
